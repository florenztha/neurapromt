

## Web3 Integration (Neura Testnet)

This workspace now includes a lightweight Web3 flow:

- **Connect wallet** via the button in the header (styled similar to PancakeSwap). Works with MetaMask, OKX Wallet, or any injected EIP‑1193 provider.
- On every prompt/AI generation request the client will ask you to confirm a payment of **0.05 ANKR** to the developer deployer wallet (`0xBdA7Ffae978bb2bC53ABc8d803D61DB768942e42`).
- If your wallet is not already on Neura Testnet it will prompt you to switch or add it automatically.

### Network parameters

- **Name:** Neura Testnet
- **RPC URL:** `https://testnet.rpc.neuraprotocol.io/`
- **Chain ID:** `267` (`0x10B`)
- **Currency symbol:** ANKR
- **Explorer:** `https://testnet-blockscout.infra.neuraprotocol.io/`
