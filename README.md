# Nabila Prompt Generator


Nabila Prompt Generator is a comprehensive AI prompt engineering toolkit for creators, marketers, and developers. Generate high‑quality prompts and results instantly—no wallets, payments, or blockchain required.

## Features

- **Content Lab**: Craft compelling captions, scripts, and marketing copy.
- **Image Studio**: Produce photorealistic images and edit photos with AI.
- **Video Factory**: Generate cinematic video descriptions and storyboards.
- **Voice Lab**: Create professional voiceovers and sound effects.
- **Reverse Engineering**: Extract powerful prompts from existing images or videos.
- **AI Assistant**: Get real-time guidance and prompt optimization.

## Getting Started

1. Clone the repo and install dependencies.
2. Set up your environment variables (e.g. `NEXT_PUBLIC_GEMINI_API_KEY`).
3. Run `npm run dev` and open http://localhost:3000 in your browser.
4. Choose a tool, configure inputs, and click **Generate Result & Prompt**.

## Technical Details

Built with Next.js 15, TypeScript, and Tailwind CSS. No Web3 or wallet integration is included. All AI calls are handled via server-side APIs with key rotation.

## Localization

Supports English and Indonesian languages.
